= Mark Category Keys
:navtitle: Mark
:source-language: yaml

The keys in the `mark` category apply to an inline mark phrase.

[cols="3,4,5a"]
|===
|Key |Value Type |Example

|font-color
|xref:color.adoc[Color] +
(default: _inherit_)
|[source]
mark:
  font-color: '#333333'

|font-style
|xref:text.adoc#font-style[Font style] +
(default: _inherit_)
|[source]
mark:
  font-style: bold

|background-color
|xref:color.adoc[Color] +
(default: `'#ff0000'`)
|[source]
mark:
  background-color: '#fcf8e3'

|border-offset
|xref:language.adoc#values[Number] +
(default: `1`)
|[source]
mark:
  border-offset: 2
|===
