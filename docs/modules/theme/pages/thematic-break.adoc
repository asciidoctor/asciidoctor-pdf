= Thematic Break Category Keys
:navtitle: Thematic Break
:source-language: yaml

The keys in the `thematic-break` category control the style of thematic breaks, which are also known as horizontal rules.

[cols="3,4,5a"]
|===
|Key |Value Type |Example

|border-color
|xref:color.adoc[Color] +
(default: `'#eeeeee'`)
|[source]
thematic-break:
  border-color: '#eeeeee'

|border-style
|`solid` {vbar} `double` {vbar} `dashed` {vbar} `dotted` +
(default: `solid`)
|[source]
thematic-break:
  border-style: dashed

|border-width
|xref:measurement-units.adoc[Measurement] +
(default: `0.5`)
|[source]
thematic-break:
  border-width: 0.5

|margin-bottom
|xref:measurement-units.adoc[Measurement] +
(default: `12`)
|[source]
thematic-break:
  margin-bottom: 18

|margin-top
|xref:measurement-units.adoc[Measurement] +
(default: `0`)
|[source]
thematic-break:
  margin-top: 6
|===
